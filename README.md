# Comentarios
Tutorial de como rodar o site localmente:

❗ Instale o [Python](https://python.org/)

# 1. Instalação dos módulos necessários
   ```cmd
   pip install -r requirements.txt
   ```
# 2. Rodar
   ```cmd
  python -m flask run
   ```

# 3. Entrar no site
- Clique [AQUI](http://127.0.0.1:5000/) para abrir a URL local
